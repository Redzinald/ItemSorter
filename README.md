# ItemSorter
Reverse and Regular sort - Images, Documents, Videos also Custom sorting.
Open .exe and you can get started!
1. Press "Open Sorter" button for sorter tab to open.
2. Once sorter tab opens you can either drag or press "browse" label to choose directory you wish to sort.
3. After uploading directory you can check boxes what you wish to sort, if you wish to rename folders check "Rename folders" box. Also you can press custom check box and choose file extensions you want it to contain.
4. In the end you can press "SORT!" button to sort files in that directory.
5. If you didn't like sort or need it to be unsorted, just press "REVERSE SORT!" with same exact names.
